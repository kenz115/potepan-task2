POTEPAN CAMP コースについて

POTEPAN CAMPでは複数のコースをご用意しており、 受講頂く方の学習状況や目的を基に最適なコースをご案内しております。

今後エンジニアとしてのキャリアを形成していきたい方

今後自分でサービスを作ったり副業で仕事を獲得したい方

プログラミングスキルを身に着けて今の仕事に活かしたい方

様々な背景を持った方にご受講いただいております。 実際のコース決定は社内のカウンセリング担当と行いますのでご安心ください。

HTML/CSSの基礎〜実践&レビュー

メモリ8GB以上のPC(Mac or Windows)を所持

学習内容

HTML / CSS

HTML / CSSの基礎学習とサイト模写などの実践課題を行い、綺麗なサイトを作れるようにします。

JavaScript

JavaScriptの基礎学習を行い、ストップウォッチや電卓などのアプリ開発とレビューを行います。

Ruby

Rubyの基礎学習を行い、メモアプリやじゃんけんアプリの開発とレビューを行います。

Ruby on Rails

Ruby on Railsの基礎を学び、マッチングサイトを開発しレビューを行います。

SQL

SQLを学び、必要なデータを出し入れ出来るようなスキルを身に付けます。

コマンドライン

コマンドラインを学び、ファイルの操作やGitなどスムーズに使えるようにします。
